TCPGame / Multiplayer Shooter Prototype

This is a fully playable prototype for a multiplayer shooter built in Java. It uses TCP/IP for connectivity.

This game is written entirely in pure Java. All art assets were drawn by myself.

Video Preview:


Current Features:
  • Complete game mechanics [weapons, spawns, projectiles, and physics]
  • Multiplayer connectivity [TCP/IP server and client]
  • Chat and scoreboard
Potential Features (in the case this moves beyond a simple prototype):
  • Levels and level editor
  • Additional game modes [teams, zombies]
  • Production graphics
